# 

关于Android，目前想总结的是比较核心的一些模块，比如显示系统、Input系统、还有Binder等。关于Binder，对于Android为什么要做一个这样的进程通信机制，以及Binder为什么适配于Android系统，和Binder实现用到的策略、设计模式、思想等方面去做总结。 

对于其他模块也同样如此，从知其然，也知其所以然的深度探究路径来展开阐述。所以从这个角度讲，我们描述的并不只是关于Android内容，更多是探究设计思想、设计模式、重要技术等方面。而较少的去关注实现细节，以及使用罗列代码的方式去讲述实现。

其中会有一些交叉内容，比如Linux的内容，c++的内容，对于这些交叉点，会做出说明，并且链接到其他系列的具体章节。
